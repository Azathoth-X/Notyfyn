# Notyfyn

Periodic notifications to keep you in control 

## Features

- Built with react 
- Custom message for notification taken to as input

## Getting Started

### Prerequisites
- npm/yarn

### Installation

```bash
# Clone repository
git clone <repository-url>

# Install dependencies
npm install

# Start development server
npm run dev
```
